# AI Aquarium 仕様書

## 起動

PowerShell で次を実行します。

```powershell
Start-Process .\boids-aquarium\index.html
```

ブラウザで直接 `C:\workspace\youtube\boids-aquarium\index.html` を開いても動きます。

## ファイル構成

```text
boids-aquarium/
  index.html   画面の骨組み
  style.css    見た目とレイアウト
  main.js      Canvas 描画とシミュレーション本体
  SPEC.md      この仕様書
```

## 画面

- 左側: Canvas の水槽シミュレーション
- 右側: 魚や生き物の数、群れの動き、速度などを変えるコントロールパネル
- 左上: タイトル、魚数、FPS 表示
- 左下: 操作ヒント

## 操作

- クリック: 魚の餌を置く
- 矢印キー: ダイバーを操作する
- Reset: 水槽を初期化する
- もう一度: 勝利後にゲームを初期化して再プレイする

## 魚と生き物

魚データは `fish` 配列で管理します。

主な種類は次の通りです。

- small: 小さい魚
- middle: 中くらいの黄色い魚
- boss: 大きい BOSS 魚
- ray: えい型
- octopus: たこ型
- squid: いか型
- mola: まんぼう型

魚は Boids 風のルールで動きます。

- alignment: 近くの魚と向きをそろえる
- cohesion: 近くの魚の集団へ寄る
- separation: 近すぎる魚から離れる
- food attraction: 餌へ向かう
- pointer avoidance: カーソル付近を避ける
- wall steering: 壁に近づきすぎたら戻る

## 数の調整

右側パネルのスライダーで、生き物の数を調整できます。

- Small Fish
- Middle Fish
- Boss Fish
- Fish Shape
- Ray
- Octopus
- Squid
- Mola

スライダーの値は `localStorage` に保存されるため、ページをリフレッシュしても設定が残ります。

## 食物循環

- クリックで通常の餌を置く
- 地面には海藻やサンゴ風の植物がある
- 子どもの魚は餌やプランクトンを食べる
- 成魚は海藻など地面の植物を食べる
- 魚が一定量食べると成魚になる
- 成魚は卵を産む
- 卵は一定時間後に孵化して子どもの魚になる
- サメの排出物からプランクトンが生まれる
- プランクトンは魚の餌になる

サメ以外の生き物は、繁殖や孵化によって上限なしで増えます。

## サメ

サメはランダムな速度を持ち、魚を追いかけて捕食します。

- 速い個体は魚を捕まえやすい
- 遅い個体は捕食しにくい
- 魚はサメに追われると逃げる
- サメがお腹いっぱいになると排出物を出す
- 排出物からプランクトンが発生する

ダイバーが出現している間、サメは本気モードになり、通常より強い加速と最高速度でダイバーを追います。

## ダイバーゲーム

ゲーム開始前に `初級`、`中級`、`上級` の難易度を選択します。

- 初級: サメが少し遅く、捕まる判定も小さめ
- 中級: 標準バランス
- 上級: サメが速く、捕まる判定も大きめ

ときどき人間のダイバーが出現します。

- ダイバーは矢印キーで操作する
- サメが近いとダイバーは逃走状態になる
- 体力ゲージは表示として残るが、体力切れで速度は落ちない
- ダイバーがサメに捕まると、赤いエフェクト、効果音、大量のプランクトン、卵の孵化が発生する
- 難易度を選んでから5秒後にダイバーが画面外から登場する
- ダイバーがキャンバス内に入った瞬間から生存時間を数える
- Canvas 上部に残り時間を表示する
- キャンバス内で20秒間生き残ると勝利
- 勝利すると Canvas 中央に `YOU WIN!` を表示する
- サメに食べられた場合は Canvas 中央に `GAME OVER` を表示する
- 勝敗が決まるとゲーム更新を止める
- `もう一度` ボタンを押すと初期化して、もう一度プレイできる

## 効果音

効果音は外部音声ファイルではなく Web Audio API で生成します。

- 餌を食べた音
- 海藻を食べた音
- サメの捕食音
- ダイバー捕食時のインパクト音
- ゲーム中の神秘的な水族館風BGM
- ゲームクリア時の明るいジングル
- ゲームクリア時の派手でかわいい結果画面
- ゲームオーバー時の残念そうな下降ジングル

ブラウザの自動再生制限に対応するため、クリックなどのユーザー操作後に音が鳴り始めます。

## 実装の中心

`main.js` の主な関数です。

- `reset()`: 水槽とゲーム状態を初期化する
- `syncFishCount()`: スライダーに合わせて生き物の数を調整する
- `updateFish(f)`: 魚1匹の動きを更新する
- `updateShark()`: サメの追跡、捕食、移動を更新する
- `updateDiver()`: ダイバーの操作、逃走、20秒勝利を更新する
- `showWinMessage()`: 勝利状態にして結果表示を出す
- `loop(time)`: 毎フレームの更新と描画を行う
