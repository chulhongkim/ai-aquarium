# Install on iPhone

AI Aquarium is now set up as a PWA. To add it to an iPhone:

1. Put the `boids-aquarium` folder on an HTTPS web host.
2. Open the hosted `index.html` URL in Safari on the iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open AI Aquarium from the new home screen icon.

Notes:

- iPhone installation must be done from Safari.
- A plain local file path is not enough for full PWA behavior.
- Service Worker caching works after the site is opened from a proper web URL.
